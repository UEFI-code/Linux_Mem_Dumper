# Very Lite Linux Memory Dumper

A easy, lite tool for user to dump a process memory just give a pid.

Proudly wrote & dbg with Vim in C, so easy to Build/Compile.

## How to build

```bash
gcc dumper.c -o dumper
```

## Usage

```bash
./dumper [pid]
```
